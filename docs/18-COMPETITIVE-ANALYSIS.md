## 1. Глобальные конкуренты

### Playtomic
**Страна:** Испания | **Рынок:** Европа, Латинская Америка | **MAU:** 2M+

| Аспект | Описание |
|--------|---------|
| Фокус | Бронирование кортов + matchmaking |
| Сильные стороны | Интеграция с клубами, оплата онлайн, огромная база кортов |
| Слабые стороны | Нет ELO рейтинга, слабое community building, нет в Казахстане |
| Монетизация | Комиссия с бронирований (15-20%), подписка клубов |
| Релевантность для нас | Низкая — другой рынок, другая модель (booking > community) |

### UTR (Universal Tennis Rating)
**Страна:** США | **Рынок:** Глобальный | **Пользователи:** 1M+

| Аспект | Описание |
|--------|---------|
| Фокус | Рейтинговая система для тенниса |
| Сильные стороны | Признанный рейтинг, интеграция с федерациями, данные матчей |
| Слабые стороны | Платный ($99/год), нет social features, нет чатов |
| Монетизация | Подписка $99/год (PRO), лицензирование рейтинга |
| Релевантность | Средняя — мы берём идею рейтинга, но добавляем social layer |

### Meetup / Sporty
**Фокус:** Организация спортивных встреч

| Аспект | Описание |
|--------|---------|
| Сильные стороны | Лёгкое создание событий, широкая аудитория |
| Слабые стороны | Не специализированы под теннис, нет рейтинга, нет match results |
| Релевантность | Низкая — слишком generic |

### Tennis Point / SwingVision
**Фокус:** Трекинг матчей, видеоанализ

| Аспект | Описание |
|--------|---------|
| Сильные стороны | AI-анализ игры, статистика ударов |
| Слабые стороны | Дорого, требует камеру, нет social/community |
| Релевантность | Низкая — другой сегмент (performance analytics) |

## 4. Наши конкурентные преимущества

### 1. Единственные в Казахстане
Нет ни одного специализированного приложения для любительского тенниса в КЗ. Мы — first mover.

### 2. Community-first (не booking-first)
Playtomic = бронирование кортов. Мы = социальная платформа вокруг тенниса. Для КЗ это важнее, т.к. бронирование часто идёт по телефону/WhatsApp.

### 3. Бесплатный рейтинг
UTR берёт $99/год. Наш ELO — бесплатный, что критично для рынка КЗ.

### 4. Локализация
Русский + казахский + английский. Ни один конкурент не поддерживает казахский.

### 5. Offline-first community
Telegram группы доказали спрос. Мы превращаем хаотичный Telegram-чат в структурированную платформу с рейтингом, ивентами и статистикой.

## 6. Go-to-Market Strategy (vs конкуренты)

### Фаза 1: Bootstrap (месяцы 1-3)
- Найти 2-3 существующих клуба/группы в Астане
- Предложить бесплатную платформу для управления
- Перевести их Telegram-аудиторию в приложение
- KPI: 100 пользователей, 2 активных сообщества

### Фаза 2: Word-of-mouth (месяцы 3-6)
- Рейтинг как вирусный механизм ("узнай свой уровень")
- Бейджи и достижения как sharing triggers
- Реферальные ивенты ("приведи друга на турнир")
- KPI: 500 пользователей, 5 сообществ

### Фаза 3: Expansion (месяцы 6-12)
- Другие города КЗ (Алматы, Караганда, Шымкент)
- Партнёрство с федерацией тенниса
- Монетизация: premium features для клубов
- KPI: 2000 пользователей, 20 сообществ

# Дополнительный функционал

## 1. Карта кортов 🗺️

### Описание
Интерактивная карта с теннисными кортами Астаны. Помогает найти где играть и используется при создании ивентов.

### Доступ
- Отдельный экран (доступен из Home → Quick Actions или из меню)
- Используется как picker при создании ивента (Шаг 5: Место)

│  │     [КАРТА с маркерами]        │    │
│  │                                 │    │
│  │  📍 NTC   📍 Tennis   📍 Mega │    │
│  │          📍 Indoor             │    │
│  │                                 │    │
│  └─────────────────────────────────┘    │
│                                         │
│  ── Список ──                          │
│  ┌─────────────────────────────────┐    │
│  │ 🏟 NTC Astana                   │    │
│  │ 📍 Кабанбай батыра, 42         │    │
│  │ 🎾 6 кортов (4 крытых)        │    │

└─────────────────────────────────────────┘

### Данные корта
- Название
- Адрес
- Координаты (для карты)
- Количество кортов (крытые/открытые)
- Покрытие (хард, грунт, ковёр)
- Стоимость аренды
- Часы работы
- Телефон
- Фото (галерея)
- Рейтинг (средняя оценка)
- Привязка к сообществу (если есть)

### Наполнение данных
- Суперадмин добавляет корты через веб-панель
- Сообщества (клубы) могут привязать свои корты при верификации
- В будущем: пользователи могут предлагать новые корты (модерация)

## 2. Друзья / Избранные игроки ⭐

### Описание
Список избранных игроков для быстрого доступа. Упрощает создание ивентов и коммуникацию.

### Добавление в друзья
- Кнопка «В друзья» на публичном профиле
- Двусторонняя связь не требуется (это «избранные», не взаимная дружба)
- Уведомление получателю: «Айдар С. добавил вас в избранные»

### Экран друзей
Доступен из профиля или через поиск игроков.

```
┌─────────────────────────────────────────┐
│  ←  Мои друзья (12)                    │
├─────────────────────────────────────────┤
│  🔍 Поиск среди друзей...             │
│                                         │
│  ┌─────────────────────────────────┐    │
│  │ 👤 Марат К.     3.5  │ 💬 🎾 │    │
│  ├─────────────────────────────────┤    │
│  │ 👤 Айгерим Н.   2.5  │ 💬 🎾 │    │
│  ├─────────────────────────────────┤    │
│  │ 👤 Дамир Н.     4.0  │ 💬 🎾 │    │
│  └─────────────────────────────────┘    │
│                                         │
│  💬 = Написать  │  🎾 = Пригласить    │
└─────────────────────────────────────────┘
```

### Интеграция
- При создании ивента → шаг «Пригласить»: показывать друзей первыми
- В списке игроков: звёздочка рядом с именем друга
- В чатах: друзья выше в списке
- В фильтрах ивентов: «Только друзья»

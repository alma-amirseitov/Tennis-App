# Таб «Сообщества» 🏛

## Описание
Каталог и управление сообществами. Четыре типа с иерархией и разными уровнями доступа.

## Типы сообществ

| Тип | Иерархия | Верификация | Кто создаёт | Особенности |
|-----|----------|-------------|-------------|-------------|
| Клуб/Школа | Уровень 1 (высший) | ✓ Галочка | По заявке → суперадмин | Логотип, расписание кортов, тренеры |
| Лига | Уровень 2 | ✓ Галочка | По заявке → суперадмин | Серии турниров, собственный рейтинг |
| Организатор | Уровень 2 | ✓ Галочка | По заявке → суперадмин | Отдельные турниры, промо |
| Группа игроков | Уровень 3 | Нет | Любой игрок | Неформальная, базовые функции |

## Уровни доступа в сообщество

| Тип доступа | Вступление | Видимость контента |
|-------------|------------|-------------------|
| Открытое | Мгновенное | Всё видно всем |
| Закрытое | По заявке → админ одобряет | Шапка видна, контент — только членам |
| Платное | Оплата подписки | Шапка видна, контент — только подписчикам |

### Внутренний таб: Лента
- Посты от администрации сообщества
- Новости, анонсы
- Фото/текст
- Лайки и комментарии

### Внутренний таб: Ивенты
- Список ивентов сообщества
- Разделение: Предстоящие / Прошедшие
- Нажатие → Детальный экран ивента

### Внутренний таб: Рейтинг
- Лидерборд участников сообщества
- Внутренний рейтинг сообщества (отдельный от глобального)
- Таблица: Позиция | Игрок | Рейтинг | Игры | W/L
- Нажатие на игрока → Публичный профиль

### Внутренний таб: Участники
- Список всех участников с ролями
- Поиск по имени
- Фильтр по роли (Админ / Модератор / Тренер / Участник)
- Для админов: кнопки управления (роль, бан)

### Внутренний таб: Чат
- Групповой чат сообщества
- Текстовые сообщения
- Возможность закрепить сообщение (для админов)
- Правила чата (по кнопке ℹ️)

### Внутренний таб: Фото
- Фотогалерея сообщества
- Админы загружают фото
- Альбомы (опционально): Турниры, Тренировки, Мероприятия
- Просмотр фото на весь экран

```
в течение 1-3 рабочих дней.
```

## Платные сообщества

### Для игрока (подписка)
- На странице сообщества: кнопка «Подписаться (X ₸/мес)»
- Экран оплаты через платёжный провайдер
- Автопродление (настройки подписки в профиле)
- Грейс-период: 3 дня после окончания подписки

### Для сообщества (настройка)
- В веб-панели: включение платного доступа
- Настройка цены (месяц / квартал / год)
- Пробный период (опционально)
- Контент для неподписчиков (что видно без подписки)

## Роли внутри сообщества

| Роль | Права |
|------|-------|
| Владелец | Все + удаление сообщества + передача владения |
| Админ | Управление участниками, ивентами, контентом, настройками |
| Модератор | Модерация чата, одобрение заявок, удаление постов |
| Тренер/Судья | Ввод результатов, создание ивентов, управление сеткой |
| Участник | Просмотр, запись на ивенты, чат, посты |

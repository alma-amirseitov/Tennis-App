## Секции профиля

### 1. Статистика
- Всего игр (singles + doubles)
- Побед / Поражений
- Win rate (%)
- Статистика по типам (singles/doubles отдельно)
- Статистика по месяцам (если достаточно данных)

### 2. Рейтинг и динамика
- Текущий глобальный рейтинг (ELO-like)
- Линейный график за последние 6 месяцев
- Позиция в рейтинге каждого сообщества
- Тренд: рост / падение / стабильный

### 3. История матчей
- Хронологический список всех матчей
- Каждая запись: W/L, соперник, счёт, дата, тип
- Фильтры: период, тип (singles/doubles), результат (W/L)
- Нажатие → детали матча

### 4. Достижения / Бейджи

| Бейдж | Условие | Иконка |
|-------|---------|--------|
| Первая победа | 1 победа | 🏅 |
| Десятка | 10 побед | 🎖 |
| Полтинник | 50 игр | ⭐ |
| Серия | 5 побед подряд | 🔥 |
| Турнирный боец | Участие в 5 турнирах | 🏆 |
| Уровень вверх | Повышение NTRP | 👑 |
| Социальный | Участие в 3+ сообществах | 🤝 |
| Ветеран | 100+ игр | 🎾 |
| Авторитет | 10+ друзей | 👥 |
| Регулярный | 4 недели подряд хотя бы 1 игра | 📅 |

- Не полученные бейджи показываются серыми с прогрессом
- Уведомление при получении нового бейджа

### 5. Мои сообщества
- Горизонтальный scroll с карточками
- Нажатие → страница сообщества
- Кнопка «Найти ещё»

### 6. Мои посты
- Последние 3 поста с превью
- «Показать все →» → полный список
- Кнопка создания нового поста

### Гранулярные настройки уведомлений
Каждый тип уведомления можно включить/выключить отдельно:
- Push + In-app (оба канала настраиваемы)
- Тихие часы (не беспокоить с 23:00 до 7:00)

### Приватность
- **Кто видит профиль**: Все / Только участники моих сообществ / Никто
- **Кто может писать**: Все / Только друзья / Никто
- **Показывать статистику**: Вкл/Выкл (если выкл — в публичном профиле нет цифр)

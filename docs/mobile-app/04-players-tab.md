## Фильтры

| Фильтр | Тип | Значения |
|--------|-----|----------|
| Уровень | Multi-select | Начинающий, Любитель, Продвинутый, Профи |
| Район | Multi-select | Список районов Астаны |
| Пол | Single-select | Все / Мужской / Женский |
| Возраст | Range slider | 14–70 лет |
| Активность | Single-select | Все / Онлайн сейчас / За последнюю неделю |
| Сообщество | Multi-select | Список моих сообществ |

## Карточка игрока

**Содержит:**
- Аватар
- Имя и фамилия
- NTRP уровень (с цветовым бейджем)
- Район/город
- Количество игр, побед, поражений
- Последняя активность
- Иконка ⭐ (если в друзьях/избранных)

**Нажатие** → Публичный профиль игрока

## Сортировка
- По рейтингу (по умолчанию)
- По активности (сначала недавно активные)
- По имени (А–Я)
- По количеству игр

## Поиск
- Поиск по имени и фамилии
- Debounce 300ms
- Минимум 2 символа для начала поиска
- Результаты обновляются в реальном времени

## Пагинация
- Infinite scroll
- Загрузка по 20 игроков
- Skeleton-загрузчики при подгрузке

- **В друзья / Убрать из друзей** → добавляет в избранное

# Таб «Главная» 🏠

## Описание
Первый экран после входа. Даёт обзор активности и быстрый доступ к ключевым действиям. Цель — за 3 секунды дать понять: что нового, ближайшие игры, и быстро создать запрос.

## Компоненты

### 1. Виджет рейтинга
- Аватар, имя, NTRP уровень (с цветовым индикатором)
- Текущий рейтинг (ELO-like число)
- Win/Loss за текущий месяц
- Динамика рейтинга (стрелка вверх/вниз + дельта)
- Нажатие → переход в Профиль → секция рейтинга

### 2. Quick Actions (быстрые действия)
Две кнопки в ряд:
- **Найти партнёра** → открывает таб Ивенты с фильтром «Ищу партнёра»
- **Создать запрос** → открывает конструктор ивента (wizard)

### 3. Ближайшие игры
- Показывает до 3 ближайших ивентов из сообществ пользователя
- Каждая карточка: тип, место, дата/время, свободные места
- Ссылка «Все →» → переход на Ивенты → Мои
- Нажатие на карточку → детальный экран ивента

### 4. Лента (два внутренних таба)

#### Таб «Новости»
Посты от сообществ, в которых состоит пользователь:
- Анонсы турниров и игр
- Результаты
- Объявления от администраторов
- Новости из мира тенниса (если публикуют сообщества)

#### Таб «Лента»
Контент от игроков:
- Посты игроков из моих сообществ (текст + фото)
- Результаты матчей (автоматические карточки после внесения счёта)
- Новые участники сообщества
- Достижения игроков (бейджи)

## Empty States

### Нет ближайших игр
- Иллюстрация: пустой корт
- Текст: «Нет запланированных игр»
- CTA: «Найти партнёра»

### Пустая лента
- Иллюстрация: пустой экран
- Текст: «Здесь пока пусто. Присоединитесь к сообществу, чтобы видеть новости»
- CTA: «Найти сообщество»

## Pull-to-Refresh
- Вся страница поддерживает pull-to-refresh
- При обновлении: перезагрузка виджета рейтинга + ближайших игр + ленты

# Веб-платформа — Обзор

## Описание
Десктопная веб-панель для управления сообществами + отдельная суперадмин-панель для владельца платформы. Основные админ-функции также дублируются в мобильном приложении.

## Технологии
- **React** (Next.js или Vite + React Router)
- **UI Kit**: Shadcn/UI или Ant Design
- **State**: React Query (для серверных данных) + Zustand (для UI state)
- **Авторизация**: та же система (телефон + OTP), shared API

## Две панели

### 1. Панель управления сообществом
**URL**: app.tennisapp.kz/dashboard
**Пользователи**: Админы, модераторы, тренеры/судьи сообществ

### 2. Суперадмин-панель
**URL**: app.tennisapp.kz/admin
**Пользователь**: Владелец платформы (суперадмин)

### Множественные админы
- У сообщества может быть неограниченное количество админов
- Владелец — всегда один (может передать)
- Каждый админ имеет полные права (кроме удаления и передачи)
- Аудит-лог: кто что сделал (для владельца)

## Переключение между сообществами
Если пользователь — админ нескольких сообществ, он может переключаться между ними через dropdown в sidebar.
